# Quiz App 🎯

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript.

## Features

- Multiple Categories
  - Python
  - General Knowledge
  - Cinema
  - Science
  - Math
  - Computer

- Score Counter
- Timer for each question
- Multiple Choice Questions
- Final Score Display
- Restart Quiz Option

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Select a category.
2. Answer the questions.
3. Check your final score.
4. Restart and try again.
